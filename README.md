# mysite-zappa


## Setup Virtual Environment in your Django app folder ##
```
virtualenv venv
```

## Activate Virtual Environment ##
```
source venv/bin/activate
```

## Install Django ##
```
pip install Django
```

## Install Zappa ##
```
pip install zappa
```

## Initialize Zappa ##
```
zappa init
```

## Deploy Zappa ##
```
zappa deploy
```
